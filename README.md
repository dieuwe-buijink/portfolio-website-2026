
# Portfolio Website (2026)

This portfolio showcases a collection of personal projects, design decisions, and the implementation work I completed to build a lightweight, responsive site.

**What’s on this site**

- Curated project gallery (`work.html`) with screenshots and descriptions of completed pieces.
- Skills and tools overview (`skills.html`) highlighting technologies I used and level of experience.
- A contact page (`contact.html`) for potential collaborators or clients.
- A small experiments page (`test.html`) where I try layout and animation ideas.

**What I built and why**

- Modular HTML pages for clear content separation and easy updates.
- A single, focused stylesheet (`style.css`) to keep design consistent and easy to maintain.
- Custom JavaScript (`site.js`, `transition.js`) for smooth navigation and animated transitions between sections — chosen to keep interactions lightweight and dependency-free.
- Image assets organized under `pictures/` and optimized for web performance.

**Design & development notes**

- Responsive layout designed for both mobile and desktop, using CSS layout techniques and media queries.
- Emphasis on fast load times and accessible, readable content.
- Transitions and micro-interactions implemented by hand to retain precise control over timing and behavior.

**Project structure (key files)**

- `index.html` — Landing page and intro
- `work.html` — Projects and case summaries
- `skills.html` — Tools, languages, and proficiency notes
- `contact.html` — Contact details / form
- `style.css` — Visual design, responsive rules
- `site.js`, `transition.js` — Interaction logic and transitions
